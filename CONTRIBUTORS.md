## 🤝 How to Become a Contributor

We welcome contributions! Here's how to get your name on this list:

### Types of Contributions

1. **Code Contributions**
   - New agent skills (custom scraping logic)
   - Performance improvements
   - Bug fixes
   - New features (API endpoints, UI enhancements)

2. **Documentation**
   - Tutorial articles
   - Use case examples
   - Translation to other languages
   - Video tutorials

3. **Testing & Validation**
   - Test on new websites (report results)
   - Security testing
   - Performance benchmarks
   - Bug reports with reproduction steps

4. **Research & Academic**
   - Cite in academic papers
   - Benchmark comparisons
   - Fine-tuning experiments
   - Dataset contributions

### Contribution Process

1. **Fork** the repository
2. **Create** a feature branch (`git checkout -b feature/amazing-feature`)
3. **Commit** your changes (`git commit -m 'Add amazing feature'`)
4. **Push** to the branch (`git push origin feature/amazing-feature`)
5. **Open** a Pull Request

### Recognition Tiers

**🥇 Major Contributors** (100+ lines of code or significant feature)
- Name, GitHub link, and contributions listed here
- Special mention in README.md
- Co-author credit for specific features

**🥈 Regular Contributors** (10-99 lines or multiple PRs)
- Name and GitHub link listed here
- Thank you in release notes

**🥉 Community Contributors** (bug reports, suggestions, testing)
- Name listed in release notes
- GitHub contribution graph credit
